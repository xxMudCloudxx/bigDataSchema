### Kafka `traffic-raw` 主题 - 数据 Schema v1.2 (12 字段)



- **Kafka Topic**: `traffic-raw`
- **数据格式**: Raw String
- **分隔符**: `\t` (Tab制表符)
- **字段总数**: 13

| **序号** | **字段名称 (Field Name)** | **数据类型** | **字段说明 (Description)** | **数据来源/转换逻辑**                                        |
| -------- | ------------------------- | ------------ | -------------------------- | ------------------------------------------------------------ |
| 1        | `GCXH`                    | String       | **序号**                   | **(已清理)** `standardize.py` 清理了前导 `'`。               |
| 2        | `XZQHMC`                  | String       | **行政区划**               | 原始数据。                                                   |
| 3        | `KKMC`                    | String       | **卡口名称**               | **(已标准化)** `standardize.py` 将中文长名称 替换为 `S325_Suini...` 这样的 Codename。 |
| 4        | `FXLX`                    | String       | **方向**                   | 原始数据。                                                   |
| 5        | `GCSJ`                    | String       | **过车时间**               | **(已替换)** `simulate.py` 替换为**当前服务器时间** (格式: `YYYY-MM-DD HH:MM:SS`)。 |
| 6        | `HPZL`                    | String       | **号牌种类**               | **(已标准化)** `standardize.py` 将 `1` 或 `13` 统一为 `01` 或 `13` 等两位字符串。 |
| 7        | `HPHM`                    | String       | **车牌**                   | **(已修复)** `preprocess.py` 修复了 `6***` 这样的无效车牌，并用“AI插补” 填充了 `NaN`。 |
| 8        | `CLPPXH`                  | String       | **车辆品牌**               | **(已提取/修复)** `preprocess.py` 提取了品牌 (如 `丰田牌`) 并用“AI插补” 填充了 `NaN`。 |
| 9        | `PROVINCE`                | String       | **(新) 归属省份**          | **(AI 扩展)** `expand_features.py` 从 `HPHM` 中提取 (如 `鲁`、`苏`)。 |
| 10       | `IS_WEEKEND`              | String       | **(新) 是否周末**          | **(AI 扩展)** `expand_features.py` 从 `GCSJ` 中计算 ( `1` = 是, `0` = 否)。 |
| 11       | `IS_PEAK_HOUR`            | String       | **(新) 是否高峰时段**      | **(AI 扩展)** `expand_features.py` 从 `GCSJ` 中计算 ( `1` = 是, `0` = 否)。 |
| 12       | `VEHICLE_TYPE_NAME`       | String       | **(新) 车型可读名称**      | **(AI 扩展)** `expand_features.py` 从 `HPZL` 映射 (如 `"小型客车"`, `"货车"`)。 |
| 13       | `FUEL_TYPE`               | String       | **(新)电车还是燃油车**     | HPZL 为 51, 52 通常代表新能源（大型/小型新能源汽车）         |
